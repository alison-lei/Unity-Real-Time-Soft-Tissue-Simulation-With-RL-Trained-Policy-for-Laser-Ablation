# Unity-Real-Time-Soft-Tissue-Simulation-With-RL-Trained-Policy-for-Laser-Ablation
Realistic physics-based real-time simulation of soft tissue in Unity environment with ablation damage animation. Utilization of ML-Agents to create RL-trained policy that autonomously guides the laser to ablate endometriosis lesion patterns.

Mass Spring System: https://drive.google.com/file/d/1iZj4VN6DT5ltEExl90AnQl1bIWH6XgHJ/view?usp=sharing
RL-trained policy autonomously guiding and turning on the laser to complete the task of ablating the red lesion pattern. Episode resets each time the task is completed: https://drive.google.com/file/d/15fN0gHBBIfRZbBwZ9k4m3GrtXU9hRI1R/view?usp=sharing
